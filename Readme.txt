В приложении предусмотрена динамическая генерация базы данных, для корректной работы необходимо изменить имя сервера
в файле Web.Config, Connectionstring с именем VetDBConnectionString, свойство DataSource

Возможны проблемы с отправкой письма с подвтерждением регистрации, по причине того, что почтовый ящик банят
за частую отправку такого рода сообщений

После запуска проекта БД будет пуста, для того, что бы заполнить её начальными данными необходимо нажать кнопку
"Создать админа (для проверяющего проект)"
